# 前言

如果您觉得软件对您有所帮助，在[Github](https://github.com/CopyTranslator/CopyTranslator)给我一个star就是对我极大的支持，欢迎介绍给朋友使用。本软件免费开源，严禁出售，欢迎转载，但请注明出处，否则构成侵权。
